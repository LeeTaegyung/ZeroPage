# 프로젝트 소개

크롬 브라우저의 시작 페이지로 설정할 커스텀 페이지입니다.
자주 사용하는 크롬 브라우저 시작 페이지에 검색바 외에 투두 리스트 기능과 시간 표시가 포함되었으면 해서 제작하게 되었습니다.

# 기능 소개

- 검색창
  - 검색시 크롬 검색으로 연결.
- 투두리스트
  - 로컬스토리지 기반
  - 드래그앤 드랍으로 투두리스트 창 이동

# 기술 스택

- 개발 언어 : TypeScript, HTML, CSS
- 라이브러리 : React
- 스타일링 : emotion
- 번들링 : Vite
- 형상 관리 : Git
- 배포 : Vercel
